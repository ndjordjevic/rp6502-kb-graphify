# RP6502 Ep20 - The 6502 calls a BBS

If you have one of these 6502 boards, you're&nbsp; going to want to do this upgrade.

More on why&nbsp;&nbsp; in a moment.

It's easy to do.

Just swap the&nbsp; Raspberry Pi Picos with upgraded version two&nbsp;&nbsp; devices.

Use a plain Pico2 for the VGA and&nbsp; a Pico2W for the RIA.

The W version includes&nbsp;&nbsp; a Wi-Fi radio, which means your 6502 can be&nbsp; connected to the internet.

With this update,&nbsp;&nbsp; you can do something you probably haven't&nbsp; done in 30 years.

Connect to a BBS.

Yes,&nbsp;&nbsp; that's ANSI colored ASCII art.

The Picocomputer&nbsp; is a collection of stubbornly useful ideas which&nbsp;&nbsp; the 6502 somehow doesn't mind.

ANSI escape&nbsp; sequences and code page 437 fonts are already&nbsp;&nbsp; in there.

As if the Pico computer always knew&nbsp; it would end up being a BBS terminal one day.&nbsp;&nbsp; Internet access also means it can fetch the time&nbsp; and even adjust for daylight savings.

Useful,&nbsp;&nbsp; though perhaps less exciting than playing a game&nbsp; of Minesweeper on a BBS with interactive ASCII.
