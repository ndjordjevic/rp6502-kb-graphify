Channel exports from the Rumbledethumps Discord server, produced by DiscordChatExporter.

## Naming convention

```
server-<channel-name>-<YYYY-MM-DD>--<YYYY-MM-DD>.txt
```

- `server` = Discord server name (e.g. `rumbledethumps`)
- `channel-name` = channel name without the `#` (e.g. `chat`)
- First date = earliest message in the file
- Second date = last message in the file
- Subsequent exports start from the day after the previous file's last date
